CSS Validation

$Provider supports web standards. All of the pages across the $Provider website have their presentation controlled using external CSS files. The W3C is responsible for creating and maintaining the CSS standard. Along with creating the CSS specification, the W3C has provided a web-based program that developers can use to verify that their CSS files precisely meet the W3C standards.

Here are links you can uee to validate all of the $Provider CSS files on our website. $Provider is proud of its support for web standards and encourages developers to use CSS and ensure that their files validate against the specification.
